View Project

Responsive CMS

The Universal Shanti project was a redesign of a site I created in 2009. I kept the design almost identical but re-coded the entire thing onto my custom responsive framework. (Build on a combination of 1140gs, html5 boiler, fittext and more) The entire project needed to be built on wordpress so the client could continue to update the blog and pages herself.

Non-Mobile First

This was one of my first responsive projects so there are very obvious flaws that I use as a learning point. Loading time on mobile view is painfully slow - the header images are very large and the site was not designed with the mobile-first methodology. This causes excessive loading times on bandwidth sensitive devices. From a functional perspective, the site is just fine. It adapts to all viewports and displays content and dynamic buttons differently for the proper use-cases.
